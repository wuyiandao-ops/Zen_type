# ZenType 新手安裝指南(零經驗也能照著做)

ZenType 是一個**離線、繁體中文**的語音輸入工具:按住鍵盤上的「右 Ctrl」講話,
放開後電腦會自動把你說的話整理成通順文字,貼到游標所在的地方。

因為它是**在你自己電腦上跑的 AI**(不上傳雲端、免月費),第一次要裝一些東西。
照著下面做,大約 20~40 分鐘(大部分時間在等下載)。

---

## 一次性安裝(只需做一次)

### 步驟 1:安裝 Python
1. 打開 https://www.python.org/downloads/ ,點黃色的「Download Python」大按鈕。
2. 執行下載的安裝檔。**最重要**:安裝畫面最下方的
   **「Add python.exe to PATH」一定要打勾**,再按 Install。
3. 裝完按 Close。

### 步驟 2:安裝 Ollama(負責「潤稿」的 AI)
1. 打開 https://ollama.com/download ,下載 Windows 版並安裝(一路下一步)。
2. 裝完後它會自動在背景執行(工作列右下角會有一個羊駝圖示)。

### 步驟 3:一鍵安裝
在本資料夾裡**雙擊 `install.bat`(安裝程式)**。它會自動:
- 安裝所有需要的套件
- 下載潤稿模型(Qwen)
- 下載語音辨識模型(約 900MB)

> 這一步會下載好幾 GB,請耐心等它跑完、看到「Install finished!」再關。
> 中途若出現紅字錯誤,通常是網路問題,重跑一次 `install.bat` 即可。

### 步驟 4(選用):設定你的專有名詞
把 `corrections.example.csv` 複製一份、改名成 `corrections.csv`,
用記事本或 Excel 打開,填入你常被聽錯的字(格式:誤聽,正確,備註)。

---

## 每天使用

雙擊 **`start.bat`(啟動程式)** → 會跳出兩個黑色小視窗(別關它們)。
等 ZenType 視窗出現 `🔥 潤稿模型已預熱完成` 後:

| 想做什麼 | 按哪個鍵 |
|---|---|
| **按住說話,放開就自動打字** | **右 Ctrl** |
| 開 / 關 AI 潤稿 | **F10** |
| 結束程式 | **Ctrl + Alt + Q** |

在任何可以打字的地方(記事本、Line、瀏覽器、Word…)都能用。

---

## 遇到問題?

| 狀況 | 怎麼辦 |
|---|---|
| 雙擊 `安裝.bat` 說找不到 Python | 步驟 1 沒勾「Add to PATH」→ 重裝 Python 並勾選 |
| 按右 Ctrl 沒反應 | 對 `start.bat` 按右鍵→「以系統管理員身分執行」 |
| 講完沒有文字出現 | 確認兩個黑視窗都還開著;SenseVoice 視窗要顯示「服務就緒(埠 8009)」 |
| 潤稿好像沒作用 | 確認工作列有 Ollama 在跑;或按 F10 看狀態 |
| 貼出來是簡體 / 有多餘空白 | 正常情況已自動處理;若仍發生請回報 |

> 進階:想打包成單一 EXE 可看 `README.md`,但一般使用不需要。
